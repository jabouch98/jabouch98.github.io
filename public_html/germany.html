Schwerin Castle

This castle is on an island where a castle has been there since the 10th century. This castle spent some time as kindergarten teachers. Now, the castle is a museum and a government builing.

Hohenzollern Castle

This castle started being built as early as the 11th century. It sits on top of a hill that it was named for. However, the castle that sits there now was built in the mid-19th century. This castle is still privately owned.

Neuschwanstein Castle

This castle was built in the late 1800s. This castle served as the inspiration for Disney's Sleeping Beauty. This is the most photographed building in Germany.
